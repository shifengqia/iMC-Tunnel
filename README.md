# imc_tunnel

>一款带图形化界面的 iMC Portal网关 连接工具

#### 安装步骤

直接从 Releases 中下载安装程序

#### 支持功能
- [x] 开机自动连接
- [x] 断线重连
- [x] 自定义网关IP及端口
- [] 多平台支持
